kQuery Reveal Modal

KRL jQuery Modal Module

The Reveal Modal module is a port of the Reveal jQuery plugin to a Kynetx Module. Reveal is a simple, fast and light weight modal that is eazy to implement. Reveal is cross-browser compatible with modem browsers (with same graceful degradation of course).

Download Latest version from GitHub

See it in action

The Reveal modal has a few options, but the default modal is really quite nice all by itself. Go ahead and give it a try.

Use the Module

use module a169x519 alias kQueryReveal
...
rule init_modal {
  select when pageview ".*"
    kQueryReveal:init();
}

Add to Ruleset

Include the use module statement in the meta section of your ruleset. Then within a rule call the init() action within the module.

<div class="kquery-wrapper">
  <div id="myModal" class="reveal-modal">
     <h1>Modal Title</h1>
     <p>Any content could go in here.</p>
     <a class="close-reveal-modal">&#215;</a>
  </div>
</div>
...
<a href="#" data-reveal-id="myModal">Click Me</a>

The Modal Markup

Simply give your modal div the class "reveal-modal" and a unique ID (the ID will be used to launch the modal). Then, add the "data-reveal-id" attribute on the anchor, when clicked the plugin matches the value of the "data-reveal-id" attribute (in this case "myModal") with an HTML element with that ID.
